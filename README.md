# myFlix Client

## Description

myFlix is a single-page, responsive web application built with **React** that allows users to explore information about movies, genres, and directors. Users can create an account, log in, and manage a list of their favorite movies.

This client-side application consumes an existing **REST API** and database created as part of the server-side of the myFlix project. Together, the client and server form a full-stack **MERN** application (MongoDB, Express, React, Node.js).

---

## Objective

The objective of this project is to build the client-side of the myFlix application using **React**, providing users with a polished interface to interact with movie data served from a RESTful API.

---

## Features

### Main View

- Displays all movies with image, title, and description
- Allows users to filter movies using a search feature
- Allows users to select a movie for more details
- Navigation to Profile view
- Logout functionality

### Single Movie View

- Displays detailed information about a movie:
  - Description
  - Genre
  - Director
  - Image
- Allows users to add or remove a movie from their favorites list

### Login View

- Allows users to log in using a username and password

### Signup View

- Allows new users to register with:
  - Username
  - Password
  - Email
  - Date of birth

### Profile View

- Displays user profile information
- Allows users to update account details
- Displays favorite movies
- Allows users to remove movies from favorites
- Allows users to deregister
